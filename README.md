# 🧶 Vintage Pattern Decoder

**Decode any vintage knitting or crochet pattern into modern terminology.**

A [FiberTools.app](https://fibertools.app) product — built on 30+ years of fiber arts expertise.

## The Problem

Thousands of vintage knitting and crochet patterns from the 1920s–1980s are freely available online, but modern crafters can't use them because:

- **British terminology differs from US** (UK "treble" = US "double crochet")
- **Old abbreviations are undecipherable** — yrn, yfwd, yon, wl fwd, rf, cm, om
- **Hook/needle sizes were numbered differently** (old UK numbers run backwards)
- **Yarn brands are discontinued** — no modern crafter knows what "Patons Doublette" maps to
- **Sizing standards have changed** since the mid-century

## Features

### Pattern Translator
Paste any vintage pattern text — the app detects and translates every vintage term to modern US equivalents instantly.

### Vintage Glossary
25+ vintage-specific abbreviations with era context, usage notes, and modern equivalents. Goes beyond standard UK/US converters.

### Hook & Needle Size Converter
Old UK numbered sizes (which run backwards!) mapped to modern US letter sizes and metric millimeters. Covers crochet hooks and knitting needles.

### Yarn Substitution Guide
- Vintage-to-modern weight chart with gauge ranges
- 17+ discontinued yarn brands with specific modern substitutes
